## 🎮 Minecraft Guilin City Walk | 桂林像素漫步 🏔️

**Author**: [@MU-ty](https://github.com/MU-ty)

### Introduction

A city walking guide website that perfectly integrates Guilin landscapes with Minecraft pixel art style and TRAE.

### Links

- **GitHub Repo**: https://github.com/MU-ty/Minecraft-Guilin-City-Walk-TRAE
- **Live Demo**: https://mu-ty.github.io/Minecraft-Guilin-City-Walk-TRAE/

### Tech Info

| Property | Value |
|----------|-------|
| Type | Web App |
| Tech Stack | JavaScript / TypeScript |

### Core Highlights

1. **🗺️ PixelMap** - Interactive radial pixel path map based on SVG, covering downtown, Yangshuo, Longji and other core scenic areas
2. **🎯 TRAE in Guilin** - Records TRAE community developer activities in Guilin (Meetup, Demo Day), supports material download and map markers
3. **🎨 MC Style UI** - Uses Press Start 2P pixel font throughout, simulating game interface with block-style buttons and textures
4. **👤 Admin System** - Steve's (admin MU-ty) personal space, supports Coffee Chat booking and GitHub deep interaction
