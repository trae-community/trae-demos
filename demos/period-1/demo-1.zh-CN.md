## 🎮 Minecraft Guilin City Walk | 桂林像素漫步 🏔️

**作者**: [@MU-ty](https://github.com/MU-ty)

### 简介

一个将桂林山水与 Minecraft 像素艺术风格与 TRAE 完美融合的城市漫步导览网站。

### 链接

- **GitHub 仓库**: https://github.com/MU-ty/Minecraft-Guilin-City-Walk-TRAE
- **在线演示**: https://mu-ty.github.io/Minecraft-Guilin-City-Walk-TRAE/

### 技术信息

| 属性 | 值 |
|------|-----|
| 类型 | Web 应用 |
| 技术栈 | JavaScript / TypeScript |

### 核心亮点

1. **🗺️ PixelMap 像素地图** - 基于 SVG 实现的交互式放射状像素路径图，涵盖市中心、阳朔、龙脊等核心景区
2. **🎯 TRAE 在桂林** - 记录 TRAE 社区在桂林的开发者活动（Meetup、Demo Day），支持资料下载与地图标记
3. **🎨 MC 风格 UI** - 全站采用 Press Start 2P 像素字体，模拟游戏界面的方块化按钮、阴影与材质
4. **👤 管理员系统** - 史蒂夫（管理员 MU-ty）的个人空间，支持 Coffee Chat 预约与 GitHub 深度互动
